$ ls --help | option that displays more information on how to use the command or program.

$ ls -F | prints the names of the files and directories in the current directory.
a trailing / indicates that this is a directory
@ indicates a link
* indicates an executable

$ pwd | You can see where you are atm 

$ cd .. | go back (to previous folder) cd without an argument will return you to your home directory

$ cd - | go back to previous folder

$ cd | with argument will move you to that directory

$ ls -F -a | -a stands for ‘show all’

$ ls -S will sort the files and directories by size

$ ls -s | will display the size of files and directories alongside the names

pressing Tab twice brings up a list of all the files starting with the given letter - press tab twice

$ mkdir | make directory

$ ls -FR | will list all nested subdirectories within a directory

$ nano draft.txt | open text editor

Ctrl+O | save data to disc (other words - safe file)

Ctrl+X | exit editor

$ touch my_file.txt | create a text file

$ ls -l | inspect the files

$ rm | delete files 

$ mv thesis/draft.txt thesis/quotes.txt | system will rename file draft to quotes
